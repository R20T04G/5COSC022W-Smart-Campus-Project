# Campus Engine - Smart Campus API

## Overview
This repository contains the backend RESTful API for the "Smart Campus" sensor and room management initiative. It is being developed as part of the Client-Server Architectures coursework (5COSC022W).

**Current Status:** Repository initialized. Initial setup and architecture planning in progress.

## Tech Stack
* **Language:** Java
* **Framework:** JAX-RS (Jakarta RESTful Web Services)
* **Build Tool:** Maven
* **Data Storage:** In-memory collections (HashMap/ArrayList)

## Planned Features
- [ ] JAX-RS API Bootstrap and configuration
- [ ] Room Management (CRUD operations with safety constraints)
- [ ] Sensor Registration and Management
- [ ] Sub-resource routing for historical sensor readings
- [ ] Custom Exception Mappers (409, 422, 403, 500)
- [ ] API Logging Filters
